## mongoDB database start: 
`cd server`
`mongod --dbpath=/data`

## Express server start:
`npm run dev`

## Metro Bundler start:
cd ../
`expo start -c`

You can sign in in bottomo of the drawer as an admin to add, edit and delete events with:
* username: username
* password: password

# Event Entry
* Title - Text
* Start Date - DateTime
* Description - Text
* Place - Text
* Image - Text - URL
